🎓 GPA Calculator (Python)

 📌 Project Overview

This project is a simple **GPA (Grade Point Average) Calculator** built using Python. It allows students to input their marks for different courses and calculates their GPA based on predefined grading criteria and credit hours.

This project is ideal for beginners who want to practice:

* Python basics
* Loops and conditional statements
* User input handling
* Basic arithmetic operations

 ⚙️ How It Works

1. The program stores a list of courses and their respective credit hours.
2. It asks the user to input marks (0–100) for each course.
3. Based on the entered marks, it assigns a GPA using a grading scale.
4. It multiplies GPA with credit hours for each course.
5. Finally, it calculates the overall GPA using the formula:

GPA = (Sum of (Grade Points × Credit Hours)) / Total Credit Hours

3. Navigate to the project folder:

4. Run the program:
